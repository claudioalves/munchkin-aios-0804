# Story 3.3 — Stepper Step 2 — Configuração Modo Épico

**Epic:** EP-03 — Fluxo de Novo Jogo
**Story ID:** EP03-S03
**Status:** Ready for Review
**Points:** 2
**Branch:** `story/3.3-stepper-step2`

## Contexto

O segundo passo do stepper permite ativar o Modo Épico. OFF = nível máximo 10, vitória em 11. ON = nível máximo 20, vitória em 21. Esta tela recebe os IDs dos jogadores do Step 1 via router state e, ao confirmar, delega a criação do jogo para o Step seguinte (Story 3.4).

## Notas Técnicas

**Toggle de Modo Épico:**
```tsx
<button
  onClick={() => setEpicMode(prev => !prev)}
  className={`relative w-12 h-6 rounded-full transition-colors ${epicMode ? 'bg-brand-emerald' : 'bg-surface-elevated'}`}
>
  <span className={`absolute top-1 w-4 h-4 rounded-full bg-parchment transition-transform ${epicMode ? 'translate-x-7' : 'translate-x-1'}`} />
</button>
```

**Limites dinâmicos:**
```tsx
<p className="font-body text-parchment-muted text-sm">
  Nível máximo: <strong className="text-parchment">{epicMode ? 20 : 10}</strong> — Vitória em <strong className="text-brand-gold">{epicMode ? 21 : 11}</strong>
</p>
```
