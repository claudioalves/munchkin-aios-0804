# Story 6.4 — Narração automática de vitória

**Epic:** EP-06 — Narrador de Voz
**Story ID:** EP06-S04
**Status:** InReview
**Points:** 1
**Branch:** `story/6.4-victory-narration`

## Contexto

Quando o `VictoryModal` é exibido, deve narrar automaticamente a vitória do jogador: `"{nome} venceu! Nível {N}!"`. A narração ocorre uma única vez ao montar o modal.

## Acceptance Criteria

- [x] AC1: `VictoryModal` narra automaticamente ao montar, sem interação do usuário
- [x] AC2: Narração usa `useTTS.speak()` com o texto `"{nome} venceu! Nível {N}!"`
- [x] AC3: Narração não se repete se `VictoryModal` re-renderizar sem desmontar (useEffect com array de dependências vazio `[]`)
- [x] AC4: `npm run lint` e `npm run typecheck` passam sem erros

## Tasks de Implementação

- [x] T1: Atualizar `VictoryModal` para consumir `useTTS` e chamar `speak()` em `useEffect(() => { ... }, [])` ao montar
- [x] T2: Executar `npm run lint` e `npm run typecheck`

## Notas Técnicas

**Texto narrado:** `\`${winnerName} venceu! Nível ${winnerLevel}!\``

**useEffect com dependências vazias** garante que a narração ocorra apenas uma vez ao montar, sem repetição por re-renders.

**Graceful degradation:** se `isSupported === false`, `speak()` é no-op — nenhum tratamento adicional necessário.
