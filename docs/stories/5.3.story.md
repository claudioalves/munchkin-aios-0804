# Story 5.3 — Sync queue offline

**Epic:** EP-05 — Persistência & Salvamento
**Story ID:** EP05-S03
**Status:** Draft
**Points:** 3
**Branch:** `story/5.3-sync-queue`

## Contexto

Quando o dispositivo perde conexão, os taps de nível não devem falhar silenciosamente — devem ser enfileirados e sincronizados assim que a conexão retornar. Esta story implementa um sync queue simples em localStorage: cada write que falha é adicionado à fila, e o hook `useSyncQueue` processa a fila ao detectar `navigator.onLine = true` via evento `online`.

## Notas Técnicas

**Estrutura da fila:**
```typescript
interface QueuedWrite {
  gamePlayerId: string;
  level: number;
  timestamp: string;
}
```

**`useSyncQueue`:**
```typescript
export function useSyncQueue() {
  const flushQueue = async () => {
    const raw = localStorage.getItem(STORAGE_KEYS.SYNC_QUEUE);
    if (!raw) return;
    const queue: QueuedWrite[] = JSON.parse(raw);
    if (!queue.length) return;
    localStorage.removeItem(STORAGE_KEYS.SYNC_QUEUE);
    for (const write of queue) {
      await updateLevel(supabase, write.gamePlayerId, write.level).catch(console.error);
    }
  };

  useEffect(() => {
    window.addEventListener('online', flushQueue);
    // Flush ao montar (caso já esteja online com fila pendente)
    if (navigator.onLine) void flushQueue();
    return () => window.removeEventListener('online', flushQueue);
  }, []);
}
```

**`useLevelUpdate` — atualização (enfileirar ao invés de rollback puro):**
```typescript
updateLevel(supabase, gamePlayerId, newLevel).catch(() => {
  if (!navigator.onLine) {
    // Enfileirar para sync posterior
    const raw = localStorage.getItem(STORAGE_KEYS.SYNC_QUEUE) ?? '[]';
    const queue: QueuedWrite[] = JSON.parse(raw);
    queue.push({ gamePlayerId, level: newLevel, timestamp: new Date().toISOString() });
    localStorage.setItem(STORAGE_KEYS.SYNC_QUEUE, JSON.stringify(queue));
  } else {
    // Online mas falhou: rollback
    updatePlayerLevel(gamePlayerId, currentLevel);
  }
});
```
