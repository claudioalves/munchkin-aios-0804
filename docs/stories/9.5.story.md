# Story 9.5 — AppHeader: destaque nos botões Compartilhar e Gráfico

## Contexto

Os botões de Compartilhar (📤) e Gráfico (📊) no `AppHeader` são apenas emojis sem fundo, com baixa visibilidade. Redesenhá-los como botões com fundo, label e cor, aumentando a descoberta e o apelo visual.

- [x] AC1: Botão "📊 Gráfico" com fundo `bg-surface-card`, borda `border-brand-gold/40` e label visível
- [x] AC2: Botão "📤 Compartilhar" com fundo `bg-surface-card`, borda `border-parchment-dim/30` e label visível
- [x] AC3: Hover states com `hover:border-brand-gold` e `hover:text-brand-gold`
- [x] AC4: Layout do header mantém alinhamento com título central e botão de voltar
- [x] AC5: Em telas pequenas (< 360px), labels se ocultam graciosamente (opcional, mas labels não devem quebrar o layout)
- [x] AC6: `npm run lint` e `npm run typecheck` passam
