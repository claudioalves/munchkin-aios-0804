# Story 9.7 — Sort mode padrão: Personalizado como primário

**Epic:** EP-09 — Melhorias de Experiência de Jogo
**Story ID:** EP09-S07
**Status:** Done
**Points:** 1
**Branch:** `master`

## Contexto

O modo de ordenação padrão era `'level-desc'` (Por Nível). O usuário prefere iniciar sempre no modo `'custom'` (Personalizado), que permite arrastar e reordenar livremente. A ordem dos botões no `SortDropdown` também foi ajustada para refletir a prioridade: Personalizado primeiro, Por Nível segundo, Aleatório terceiro.

## Acceptance Criteria

- [x] AC1: Estado inicial do store: `sortMode: 'custom'`
- [x] AC2: `clearGame()` reseta para `'custom'` (não `'level-desc'`)
- [x] AC3: `SortDropdown` exibe botões na ordem: Personalizado → Por Nível → Aleatório
- [x] AC4: TypeCheck e lint passam

## Tasks de Implementação
