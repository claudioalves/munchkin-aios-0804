# Story 7.1 — Timer de snapshots e getSnapshots

**Epic:** EP-07 — Gráfico de Evolução
**Story ID:** EP07-S01
**Status:** InReview
**Points:** 2
**Branch:** `story/7.1-snapshot-timer`

## Contexto

`useAutoSave` (EP-05) já captura snapshots periódicos a cada 15 minutos. Esta story complementa o sistema com: (1) captura de snapshot inicial ao entrar na tela de partida, e (2) implementação de `getSnapshots()` para buscar o histórico de snapshots — função necessária para o gráfico de evolução (Story 7.2).

## Notas Técnicas

**Query de getSnapshots:**
```sql
SELECT * FROM level_snapshots
WHERE game_id = gameId
ORDER BY captured_at ASC
```

**Assinatura da função:**
```typescript
async function getSnapshots(supabase: SupabaseClient, gameId: string): Promise<LevelSnapshot[]>
```

**Snapshot inicial no GamePage:** usar `captureSnapshot` (já existente) dentro de `useEffect` com ref guard para execução única.
