# Story 4.4 — Detecção de vitória

**Epic:** EP-04 — Tela de Partida — Core
**Story ID:** EP04-S04
**Status:** Draft
**Points:** 2
**Branch:** `story/4.4-victory`

## Contexto

Quando um jogador atinge `victory_level` (11 em modo normal, 21 em modo épico), o jogo termina com vitória. Esta story implementa o modal de vitória: animação de celebração, nome do vencedor, e botão para encerrar a partida (que chama `finishGame()` e volta ao menu).

## Acceptance Criteria

- [ ] AC1: Quando qualquer jogador atinge `victory_level`, o modal de vitória é exibido automaticamente
- [ ] AC2: O modal exibe: "🏆 Vitória!", nome do vencedor e sua cor
- [ ] AC3: O modal tem botão "Encerrar Partida" que chama `finishGame()`, limpa o store e navega para `/`
- [ ] AC4: Durante a chamada ao Supabase, botão fica em loading
- [ ] AC5: Fundo do modal é backdrop `bg-black/80` bloqueando interação com o grid
- [ ] AC6: `npm run lint` e `npm run typecheck` passam sem erros

## Tasks de Implementação
