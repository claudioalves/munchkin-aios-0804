# Story 4.5 — AppHeader da partida

**Epic:** EP-04 — Tela de Partida — Core
**Story ID:** EP04-S05
**Status:** Draft
**Points:** 2

## Contexto

O AppHeader exibe no topo da tela de partida: título "Munchkin", modo da partida (Normal/Épico) e contador de jogadores. Também contém o `HoldButton` para encerrar a partida. Esta story implementa o header e integra com a GamePage.

## Acceptance Criteria

- [ ] AC1: AppHeader exibe título, modo (Normal / Épico ★) e número de jogadores
- [ ] AC2: AppHeader integrado no topo do `GamePage`
- [ ] AC3: Link/botão discreto de voltar ao menu (sem encerrar a partida) — apenas navega para `/`
- [ ] AC4: `npm run lint` e `npm run typecheck` passam sem erros

## Tasks de Implementação

- [ ] T1: Criar `apps/web/src/components/AppHeader/AppHeader.tsx`
- [ ] T2: Integrar `AppHeader` em `GamePage`
- [ ] T3: Executar `npm run lint` e `npm run typecheck`
