## Contexto

Para evitar encerramento acidental, a partida só termina após o usuário segurar o botão por 3 segundos contínuos. Durante o hold, uma barra de progresso preenche da esquerda para a direita. Soltar antes cancela. Ao completar, `finishGame()` é chamado.

## Notas Técnicas

**Mecânica do hold:**
```typescript
const HOLD_DURATION = 3000;
const intervalRef = useRef<ReturnType<typeof setInterval> | null>(null);
const startTimeRef = useRef<number>(0);
const [progress, setProgress] = useState(0);

const startHold = () => {
  startTimeRef.current = Date.now();
  intervalRef.current = setInterval(() => {
    const elapsed = Date.now() - startTimeRef.current;
    const p = Math.min(elapsed / HOLD_DURATION, 1);
    setProgress(p);
    if (p >= 1) {
      clearInterval(intervalRef.current!);
      onComplete();
    }
  }, 16); // ~60fps
};

const cancelHold = () => {
  if (intervalRef.current) clearInterval(intervalRef.current);
  setProgress(0);
};
```

**Visual da barra de progresso:**
```tsx
<div className="absolute inset-0 bg-brand-ruby origin-left transition-none"
     style={{ transform: `scaleX(${progress})` }} />
```
