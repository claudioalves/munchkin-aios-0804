## Notas Técnicas

**Texto narrado:** ordenar `gamePlayers` por `level` desc, gerar `"{nome} nível {level}"` para cada jogador, unir com `". "`, e passar o resultado para `speak()`.

**Toggle de comportamento:**
```typescript
isSpeaking ? stop() : speak(narratorText);
```

**Ícone:** emoji `🎙️` ou texto "Narrar" (escolher o que melhor se encaixa visualmente no layout do GamePage).

**Tooltip quando desabilitado:** `"Narrador de voz não suportado neste dispositivo"`.
