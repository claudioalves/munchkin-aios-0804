# Story 6.3 — Modal de Configurações — Voz

**Epic:** EP-06 — Narrador de Voz
**Story ID:** EP06-S03
**Status:** InReview

## Contexto

Página de configurações acessível via botão "Configurações" na Home. Permite ao usuário selecionar a voz do narrador entre as vozes disponíveis no dispositivo. A seleção persiste via `useTTS` no localStorage.

## Acceptance Criteria

- [x] AC1: Rota `/settings` existe com `SettingsPage` renderizado
- [x] AC2: Lista de vozes disponíveis exibida como `<select>` ou radio group; label inclui nome da voz e idioma
- [x] AC3: Seleção de voz chama `setVoice(uri)` do `useTTS` e persiste no localStorage
- [x] AC4: Botão "Preview" toca um exemplo de narração com a voz selecionada via `speak()`
- [x] AC5: Link ou botão "Voltar" navega de volta para a Home (`/`)
- [x] AC6: `npm run lint` e `npm run typecheck` passam sem erros
