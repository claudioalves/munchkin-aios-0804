# Story 9.6 — Vozes de Celebridades (Bolsonaro, Lula, Dilma)

## Contexto

O narrador atual usa a Web Speech API (vozes do sistema). Para adicionar vozes personalizadas de figuras públicas, é necessário uma API de síntese de voz com clonagem. Esta story documenta as opções arquiteturais e o caminho de implementação recomendado.

## Análise Arquitetural

### Por que a Web Speech API não é suficiente

A `SpeechSynthesis` API usa apenas as vozes instaladas no OS/browser. Não é possível adicionar vozes personalizadas via JavaScript — apenas selecionar entre as disponíveis.

### Opções

| Opção | Pros | Contras | Custo |
|-------|------|---------|-------|
| **ElevenLabs** | Qualidade excelente, API simples, clonagem de voz | Pago (após ~10k chars/mês grátis), latência ~300ms | $1/1M chars |
| **PlayHT** | Clonagem de voz, vozes BR | Pago, API mais complexa | $0.60/1M chars |
| **Clips de áudio pré-gravados** | Zero latência, sem custo de API, offline | Requer gravar clips para cada frase, limitado | Grátis |
| **Kokoro / Coqui TTS** | Open source, self-hosted | Complexidade de deploy, sem vozes prontas de BR políticos | Grátis |

### Recomendação: Clips de Áudio Pré-gravados

Para o contexto do Munchkin (frases curtas e específicas: "Nível X!", "Você venceu!", "Missão cumprida!"), a abordagem de clips de áudio é a mais prática:

1. Coletar clips reais ou usar TTS para frases específicas do jogo
2. Mapear cada "voz" a um conjunto de arquivos `.mp3` em `apps/web/public/voices/{voice-id}/`
3. No `useTTS`, detectar vozes customizadas e usar `new Audio(url).play()` ao invés da Web Speech API

### Estrutura de Arquivos

```
apps/web/public/voices/
  bolsonaro/
    nivel-{1..20}.mp3     # "Nível X!"
    vitoria.mp3            # "Você venceu!"
    narrar.mp3             # Frase genérica de narração
  lula/
    nivel-{1..20}.mp3
    vitoria.mp3
  dilma/
    nivel-{1..20}.mp3
    vitoria.mp3
```

### Alternativa: ElevenLabs

Se os clips forem inviáveis, integrar ElevenLabs:
1. Criar conta ElevenLabs, obter `VITE_ELEVENLABS_API_KEY`
2. Clonar vozes usando samples de ~3min de áudio público
3. Substituir a chamada `speechSynthesis.speak()` por `fetch('https://api.elevenlabs.io/v1/text-to-speech/{voice_id}')`
4. Reproduzir blob de áudio via `URL.createObjectURL()` + `new Audio()`
