# Story 9.4 — Botão de dado animado (1-6)

**Epic:** EP-09 — Melhorias de Experiência de Jogo
**Story ID:** EP09-S04
**Status:** InProgress
**Points:** 3
**Branch:** `story/9.4-dice-button`

## Contexto

Adicionar um botão de dado na tela de jogo. Ao clicar, exibe uma animação de rolagem e depois mostra o resultado (número 1-6 + face Unicode do dado ⚀⚁⚂⚃⚄⚅). Útil para decisões durante o jogo (quem começa, desempate, etc.).

## Acceptance Criteria

- [x] AC1: Botão com ícone 🎲 aparece na barra de ações inferior da `GamePage`
- [x] AC2: Ao clicar, executa animação `dice-roll` de ~700ms (rotação + escala)
- [x] AC3: Após animação, exibe face do dado (⚀-⚅) correspondente ao resultado
- [x] AC4: Exibe o valor numérico (1-6) ao lado da face do dado
- [x] AC5: Botão desabilitado durante animação (sem double-click)
- [x] AC6: Resultado persiste até próxima rolagem
- [x] AC7: Animação `dice-roll` definida em `globals.css` via `@keyframes`
- [x] AC8: `npm run lint` e `npm run typecheck` passam

## Tasks de Implementação

- [x] T1: Criar `apps/web/src/components/DiceButton/DiceButton.tsx`
- [x] T2: Adicionar `@keyframes dice-roll` e `--animate-dice-roll` em `globals.css`
- [x] T3: Integrar `DiceButton` na barra de ações da `GamePage.tsx`

## Notas Técnicas

- Faces Unicode do dado: `['⚀','⚁','⚂','⚃','⚄','⚅']`
- `Math.floor(Math.random() * 6) + 1` para resultado
- `setTimeout` de 700ms para a animação completar antes de mostrar resultado
