# Story 9.2 — Visualizador de Regras com busca (Ctrl+F)

**Epic:** EP-09 — Melhorias de Experiência de Jogo
**Story ID:** EP09-S02
**Status:** InProgress
**Points:** 5
**Branch:** `story/9.2-rules-viewer`

## Contexto

As regras do Munchkin estão divididas em arquivos Markdown dentro de `apps/web/public/regras-munchkin/`. A tela de Regras exibe o conteúdo completo com um campo de busca que destaca e navega entre ocorrências (comportamento análogo ao Ctrl+F do browser). O botão "Regras" no menu principal é habilitado. Um botão de Regras também aparece na tela de jogo.

## Notas Técnicas

- Arquivos de regras ficam em `apps/web/public/regras-munchkin/` (servidos como estáticos)
- `manifest.json` lista os arquivos na ordem de exibição: `[{ "id": "01", "title": "...", "filename": "..." }]`
- Para adicionar novas regras: add arquivo .md na pasta + entry no manifest.json
- Markdown simples renderizado como texto com estilos inline (sem dependência externa)
